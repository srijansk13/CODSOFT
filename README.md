# 🚀 Career Copilot – Premium SaaS Landing Page

A futuristic and visually rich SaaS-inspired landing page built using **HTML5** and **CSS3** as part of the CODSOFT Web Development Internship.

This project focuses on creating a modern frontend experience with advanced UI/UX design, responsive layouts, smooth animations, glassmorphism effects, and interactive visual components — all developed without using any frontend frameworks or JavaScript libraries.

# 📌 About The Project

Career Copilot is a concept-based SaaS landing page inspired by modern AI startup websites and resume analysis platforms.

The goal of this project was to design a premium-looking responsive landing page that demonstrates frontend development skills, creative UI design, visual hierarchy, animation techniques, and modern web layout practices using only HTML and CSS.

The website includes multiple professionally designed sections such as:

* Hero section
* Interactive ATS scanner simulation
* Features showcase
* Workflow timeline
* Pricing cards
* Testimonials
* Call-to-action sections
* Responsive footer

This project was developed to simulate a real-world startup landing page experience while maintaining clean structure and responsive behavior across all devices.
